# Bejibun CLI
An installation cli for bejibun framework.

## Usage

### Installation
If you don't have bun installed :

```bash
# Linux / Mac OS
curl -fsSL https://bun.sh/install | bash

# Windows
powershell -c "irm bun.sh/install.ps1 | iex"
```

Setup the project.

```bash
bunx @bejibun/cli your-project
```
